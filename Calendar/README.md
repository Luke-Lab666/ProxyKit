# Calendar

中国大陆节假日、调休、中国传统节日、二十四节气、中国常用纪念日订阅日历。

> 官方放假和调休以国务院办公厅发布为准；未来年份发布后，把对应年份 `calendar-data*.json` 放到 `Calendar/`，Action 会自动生成新的 `.ics`。
